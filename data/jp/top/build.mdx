# ビルド

Spear プロジェクトをビルドする方法は簡単です。 `yarn build` コマンドを実行するだけです。

```bash
# yarn install\n
# yarn build\n
....\n
[Parse Pages]\n
  [Page]: index(/)\n
\n
[Page]: articles/index\n
\n
[Page]: /index\n
[Sitemap]: /sitemap.xml\n
```

生成されるディレクトリは `dist` がデフォルトです。このディレクトリを確認してください。


## 開発モード

Spear にはホットリロード機能を持つ開発モードをがあります。もし、`yarn dev` コマンドを実行したら Spear は開発モードで起動します。

```bash
# yarn dev\n
...\n
[Parse Pages]\n
  [Page]: index(/)\n
\n
[Page]: articles/index\n
\n
[Page]: /index\n
\n
    Server started on port 8080 🚀\n
    You can access the following URL:\n
\n
      http://localhost:8080\n
```

[http://localhost:8080](http://localhost:8080/) にアクセスすることで、生成されたサイトを確認できます。

注意：
- もしマシン上の 8080 ポートが使われている場合、開発モードは失敗します。もし `8080 is already in use` エラーに遭遇したら、ローカル環境を確認してください。
  - この場合、設定ファイルでポートを変更することができます。詳細は、[インストールドキュメント](/{{lang}}/intro/install.html)を参照してください。
