# はじめるには

## Spear とは？

Spear は簡単にビルドができる Web のフレームワークです。

## 主要な機能

- **コンポーネント**: Spear はコンポーネント機能を提供しています。
- **ヘッドレス CMS との接続**: Spear は Spearly CMS と言ったヘッドレス CMS と接続しコンテンツを取得できます。
- **ノープログラム**: プログラム動作するようなランタイムは必要ありません。
- **プラグイン**: Spear は機能を拡張するためにいくつかのプラグインを提供しています。

## 使ってみましょう

Spear のはじめとして、実際に使ってみることが一番早いと思います。以下の StackBlitz のエディタを開いてみたください。

[![stackblitz](https://developer.stackblitz.com/img/open_in_stackblitz.svg)](https://stackblitz.com/edit/stackblitz-starters-p72h35?file=src%2Findex.html)

## プロジェクトを始める

以下のコマンドを実行するだけで Spear プロジェクトをローカルに作成できます。

```
> yarn create spear
```

## Spear を学ぶ

Spear のためのドキュメントはいくつかあります。以下のページかナビゲーションからアクセスしてください。

🖊 プロジェクト構造 - Spear の基本的なプロジェクト構造を学べます。  
🖊 コンポーネント - 他のフレームワークにもあるコンポーネントファイルを利用できます。  
🖊 データのフェッチ - Spearly からデータをフェッチするのはとても簡単にできます。

## Spear の拡張

- Spear は [プラグイン機能](./plugins/)があり、用意に拡張できます。
- もちろん、Spear OSS へ参加いただくこともできます。
