# Spear をはじめる

## Spear とは?

Spear は簡単に利用できる Web フレームワークです。

## 主要機能

- **コンポーネント**: Spear はコンポーネント機能を提供しています。
- **ヘッドレスCMSとの接続**: Spearは Spearly CMS に接続することができます。
- **プログラムレス**: プログラムランタイムは必要ありません。
- **プラグイン**: Spear は提供しているプラグインを利用して機能を拡張できます。

## 使ってみましょう

Spear を簡単に始める方法は実際に動かして機能を使ってみることです。StackBlitz のエディタを開いてみましょう。

### [![stackblitz](https://developer.stackblitz.com/img/open_in_stackblitz.svg)](https://stackblitz.com/edit/stackblitz-starters-p72h35?file=src%2Findex.html)


## 初めてのプロジェクトを作る

Spear のプロジェクトを作って時間をかけずにローカルで実行することができます。

```
> yarn create spear
```

## Spear を知る

Spear のドキュメントを提供しています。以下のページまたはナビゲーションから情報を探すことができます。

🖊 プロジェクト構造 - Spear プロジェクト構造の基本的な事を知ることができます。
🖊 コンポーネント - 他の Web フレームワークのようなコンポーネントを利用することができます。
🖊 データのフェッチ - Spearly からデータをフェッチするのは簡単です。

## Spear を拡張する

- Spearは [プラグイン機能](/{{lang}}/basic/plugin.html)があり、機能を拡張することができます。
- [Spear OSS](https://github.com/unimal-jp/spear) に貢献することも可能です！
