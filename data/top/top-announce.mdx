# Getting Started

## What is Spear?

Spear is an web framework for building which is easy to use.


## Key Features

- **Components**: Spear provide copmoonents feature.
- **Headless CMS Conectivity**: Spear can connect with Spearly CMS and Markdown.
- **No Program** code*: No Programatical runtime.
- **Plugins**: Spear has several plugins for expanding feature.

## Try it!

The spear starter is the fastest way to explore the Spear feature. Open it in StackBlitz editor:

[![stackblitz](https://developer.stackblitz.com/img/open_in_stackblitz.svg)](https://stackblitz.com/edit/stackblitz-starters-p72h35?file=src%2Findex.html)

## Start first project

You can create Spear project and run locally without many time.

```javascript
yarn create spear
```
